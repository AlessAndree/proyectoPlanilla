Bienvenido

La aplicación Web Planilla GT, es un sistema elaborado como parte del proyecto de tesis del alumno: Andreé Alessandro Espinoza. Que forma parte de la Universidad Mariano Gálvez de Guatemala con sede en Guastatoya.

Esta aplicación tiene el objetivo de facilitar a las micro y pequeñas empresas de Guatemala el uso de un sistema para gestionar el pago de planilla de las mismas.

Por motivos de sostenibilidad, la aplicación contará con un periodo de prueba gratuito, empezando desde el día de lanzamiento hasta el día 1 de diciembre del año 2,022 (La fecha puede variar según el uso del sistema). Donde empezará a tener un precio el uso del sistema.

El precio final cómo el sistema estarán sujetos a cambios, todo lo relacionado a esto será notificado con tiempo.
